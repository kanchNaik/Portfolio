    Cancer Pathology MLLM

    A multimodal research project integrating Vision Transformers and Large Language Models for cancer pathology image classification. The system combines caption-guided instruction tuning, retrieval-augmented generation (RAG), and advanced prompting strategies (Zero-Shot, Few-Shot, Chain-of-Thought, Tree-of-Thought) to balance accuracy with interpretability in medical AI diagnostics.


    • Overview: Multimodal framework for cancer pathology diagnosis using histopathology images + medical text.
    • Techniques: Caption-guided instruction tuning (BLIP2 + GPT-2), RAG with dual-channel fusion (image + text), CoT & ToT prompting.
    • Datasets: PatchCamelyon (PCam), PatchGastricADC22, BIOMEDICA.
    • Performance:
      • ViT + LoRA: 87% Accuracy, 87.5% Recall
      • GPT-4o + CoT: 74% Accuracy
      • RAG + CoT: up to 88.6% Recall
    • Technologies: Python, PyTorch, CLIP, BLIP2, GPT-2, FAISS, LLaVA, GPT-4o.
    • Features: Explainable AI (captions + reasoning), Visual + Text retrieval, Low-resource fine-tuning with LoRA, Clinical interpretability.

    Agentic AI: Multi-Agent Code Generation System

    A modular, agent-based AI architecture designed to convert natural language requests into validated frontend + backend code using API documentation and retrieval-augmented generation (RAG). Built to simulate intelligent coordination among agents and support iterative feedback loops.


    • Key Agents: Instruction Parser, Retriever (RAG), Code Generator, Code Validator, Feedback Handler, Conversation State Manager
    • RAG Setup: Uses BGE-small-en for embedding API docs (from CSV), stored in FAISS + metadata dictionary
    • Code Generator: Combines user intent, subtasks, and retrieved API docs to generate full-stack code using GPT-4 (or Mistral/LLaMA)
    • Validation Layer: Syntax check, linting, API match, and security screening before user delivery
    • Feedback Loop: User rejection triggers dynamic re-prompting using stored session history, enhancing regeneration accuracy
    • Tech Stack: Python, FAISS, Sentence-BERT, BeautifulSoup, Playwright, GPT-4, Mistral, Pydantic
